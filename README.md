# 🚀 Python Mini-Projects Portfolio

Welcome to my repository! This is a collection of my practical Python projects focused on API integration, data processing, and business logic implementation. 

## 📂 Projects Overview

### 1. 🐈 Cat Encyclopedia 2.0 (Desktop GUI App)
A fully functional desktop application built with PyQt5 that serves as an interactive encyclopedia for cat breeds. 
* **How it works:** The application features a custom-styled graphical user interface (GUI). Users can search for specific cat breeds. The app validates the input ("defense line" validation) and makes asynchronous calls to a public Cat API. It parses the JSON data to display breed characteristics and automatically logs/saves the search results into local text files.
* **Key Skills:** Python, PyQt5 (Desktop UI/UX design with CSS styling), REST API requests, JSON parsing, error handling, and File I/O operations.

### 2. 🌤️ Real-Time Weather API App
An application that fetches and displays real-time weather data for any location.
* **How it works:** The user enters the name of a city into the input field. The application makes a request to a Weather API, retrieves the current meteorological data, and instantly displays the current weather conditions on the screen.
* **Key Skills:** REST API integration, JSON parsing, user input handling.

### 3. 🎮 Game Currency to EUR Converter
A financial calculator script designed to convert in-game virtual currency into real-world Euros (EUR), accounting for in-game market fees.
* **How it works:** The application features two input fields:
  1. The amount of in-game currency the user holds.
  2. The current exchange rate of the game currency to 1 EUR.
* **Business Logic:** When the user clicks the "Calculate" button, the script applies the following formula:
  `Final EUR = (In-Game Currency * 0.8) * Exchange Rate`
  *(Note: The `0.8` multiplier accounts for a standard 20% in-game market commission fee).*
* **Key Skills:** Algorithmic logic, mathematical operations, GUI/CLI interaction.
